
# Edu Craft MAIT

This repository contains the source code for the Edu Craft MAIT project.

## Getting Started

To get started with the project, follow these steps:

git clone https://github.com/anshu-official/Edu_Craft_MAIT_.git

cd Edu_Craft_MAIT_


## Frontend
cd client_new
npm start

## Backend
cd ../server_new
npm run dev
